### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 个人中心、3. 学生信息管理、4. 公寓资产管理、5. 缴费信息管理、6. 清理公共场所管理、7. 日常事务管理、8.床位安排

管理员功能：1. 登录、2. 个人中心、3. 学生管理、4. 宿管员管理、5. 管理员管理、
